## Enforcement

Violations of this Code of Conduct will result in:

1. **Warning** - First offense, educational guidance
2. **Temporary Ban** - Repeated violations or serious misuse
3. **Permanent Ban** - Malicious use or illegal activity
4. **Law Enforcement Notification** - Serious criminal activity

## Reporting

To report violations:
- Open a GitHub issue (for minor violations)
- Contact project maintainers privately (for serious concerns)
- Report to appropriate authorities (for illegal activity)
